# Home-SOC-Lab

## 🚀 Propósito
Desarrollo de un entorno SOC de nivel profesional enfocado en la visibilidad de eventos, correlación de logs y respuesta ante incidentes, optimizando recursos mediante virtualización y gestión de endpoints.

## 🛠️ Infraestructura
Implementación de un servidor Wazuh (All-in-one) centralizando las capacidades de seguridad en un nodo de gestión:
- **Wazuh Manager:** Motor de correlación y análisis de logs.
- **Wazuh Indexer:** Almacenamiento seguro.
- **Wazuh Dashboard:** Interfaz web de monitoreo.
